# Brand Switching Analysis using Markov Chains and ANOVA

## Overview
This project models consumer brand switching behavior using Markov chains. It predicts future preferences and analyzes long-term market trends.

## Features
- Transition matrix computation
- Multi-step prediction (Markov Chains)
- Steady-state (long-term market share)
- Convergence analysis
- ANOVA for factor influence

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Statsmodels
- Streamlit

## How to Run

pip install -r requirements.txt
